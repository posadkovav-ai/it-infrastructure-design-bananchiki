# Лабораторная среда: VirtualBox + Active Directory

## Цель
Развернуть изолированный домен Windows для трёх виртуальных машин, создать пользователя и организовать общий доступ к папке.

## Использованные технологии
- Oracle VirtualBox 7.x
- Windows Server 2019 (контроллер домена)
- Windows 8.1, Windows 7 (клиенты)
- Внутренняя сеть VirtualBox (без доступа в интернет)

## Схема сети

| Устройство | IP-адрес | Роль |
|------------|----------|------|
| DC (Server) | 192.168.0.1 | Контроллер домена, DNS, общая папка |
| Win7       | 192.168.0.2 | Клиент, введён в домен |
| Win8       | 192.168.0.3 | Клиент, введён в домен |

Все машины подключены к одной внутренней сети `intnet`.

## Что сделано (по шагам)

1. **Настройка VirtualBox** – для всех трёх ВМ выбран тип «Внутренняя сеть», имя сети `intnet`.  
   ![Настройка сети](images/vbox_internal_network.png)

2. **Назначение статических IP** – на каждой ВМ прописан свой адрес из подсети 192.168.0.0/24, шлюз не указан (изолированная среда).  
   ![Статический IP на Win7](images/static_ip_win7.png)

3. **Переименование компьютеров** – клиентам даны понятные имена `WS-7` и `WS-8`.  
   ![Переименование](images/rename_pc.png)

4. **Установка Active Directory** – на сервере установлена роль AD DS, создан новый лес `POSADKOVAGORDEEVA.kursovaya`.  
   ![Создание домена](images/ad_domain_setup.png)

5. **Создание пользователя** – добавлен пользователь `Виктория Посадкова` (логин `v.posadkova`), пароль, отключена смена пароля при входе.  
   ![Новый пользователь](images/ad_new_user.png)

6. **Ввод клиентов в домен** – Windows 7 и 8 присоединены к домену через свойства системы.  
   (на скриншоте: окно «Присоединение к домену» с именем домена)

7. **Общая папка** – на сервере создана папка `C:\Обмен`, открыт общий доступ для всех (чтение/запись). Клиенты видят её по сети.  
   ![Доступ к общей папке](images/shared_folder_access.png)

## Результаты проверки

- Пинг с клиента до сервера успешен:  
  ![Пинг](images/ping_test.png)
- Из проводника клиента открывается `\\192.168.0.1\Обмен`, можно создать/удалить файл.

## Как воспроизвести самостоятельно

1. Установи VirtualBox.
2. Создай три ВМ: Server 2019, Win8.1, Win7.
3. Для каждой настрой сеть: «Внутренняя сеть» → имя `intnet`.
4. В каждой ВМ вручную задай IP из таблицы выше.
5. На сервере: добавь роль AD DS, создай лес `твойдомен.kursovaya`.
6. Добавь пользователя.
7. Клиенты переименуй и введи в домен (при вводе учётки используй созданного пользователя).
8. На сервере создай папку, открой общий доступ.
9. Проверь пинг и доступ к папке с клиентов.

- управление пользователями и группами;
- организация общего доступа к файлам в домене.
