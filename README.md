# Лабораторная среда: VirtualBox + Active Directory

## Цель
Развернуть изолированный домен Windows для трёх виртуальных машин, создать пользователя и организовать общий доступ к папке.

## Использованные технологии
- Oracle VirtualBox 7.x
- Windows Server 2019 (контроллер домена)
- Windows 8.1, Windows 7 (клиенты)
- Внутренняя сеть VirtualBox (без доступа в интернет)

## Схема сети

| Устройство | IP-адрес | Роль |
|------------|----------|------|
| DC (Server) | 192.168.0.1 | Контроллер домена, DNS, общая папка |
| Win7       | 192.168.0.2 | Клиент, введён в домен |
| Win8       | 192.168.0.3 | Клиент, введён в домен |

Все машины подключены к одной внутренней сети `intnet`.

## Как воспроизвести самостоятельно

1. Установи VirtualBox.
2. Создай три ВМ: Server 2019, Win8.1, Win7.
3. Для каждой настрой сеть: «Внутренняя сеть» → имя `intnet`.
4. В каждой ВМ вручную задай IP из таблицы выше.
5. На сервере: добавь роль AD DS, создай лес `твойдомен
6. Добавь пользователя.
7. Клиенты переименуй и введи в домен (при вводе учётки используй созданного пользователя).
8. На сервере создай папку, открой общий доступ.
9. Проверь пинг и доступ к папке с клиентов.
