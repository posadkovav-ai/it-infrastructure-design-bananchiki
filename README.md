# it-infrastructure-design-bananchiki
Проект по проектированию ИТ-инфраструктуры с нуля для компании, которая разрабатывает и продаёт умные термостаты, розетки и датчики качества воздуха. Цель - создать гибридную инфраструктуру (облако + локальный сервер), обеспечить безопасность, сегментацию сети и поддержку удалённой работы.
